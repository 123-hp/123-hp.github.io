HP Laserjet Pro M477fdw Printer – Paper jam

  • In case the jammed paper is in the interior of the printer,clear it and then try to print the configuration page.
  • Examine that the tray is set to the correct paper size. Modify the settings of the paper,if required.
  • Try to reboot the system,switch off the device,wait for a minute and then on it on again.
  • Make an attempt to print the cleaning page to discard the excess from the interior of the product.
  • In case you view any jammed paper in the document feeder,you need to discard them from the HP Laserjet Pro M477fdw printer.
  • Raise the document feeder cover. Eliminate the jammed paper.Seal the scanner lid.try to print the test page.
  • Raise the document feeder input tray to provide the access to the document feeder output bin.
  • You need to eradicate the jammed paper from the output area.Raise the scanner lid.
  • In case there is any jammed paper in the scanner lid,smoothly drag it out from the printer to avoid ripping of papers.
  • After eliminated the jammed papers,lower the scanner lid. Check for any other jammed papers in Trays,rear door and fuser area.
  • Make sure that while eliminating the jammed paper in the fuser area,check whether the fuser is cool and then remove it.
  • While doing so,check whether the carriage in the printer moves smoothly without any hindrances.
  • If you any hindrances are visible,remove them from the carriage path and then try to print a test page.
